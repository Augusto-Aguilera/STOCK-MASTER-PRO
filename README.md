**STOCK-MASTER PRO** es un ecosistema de gestión de inventario inteligente diseñado para optimizar el control de existencias en entornos industriales y comerciales. Desarrollado bajo el sello de calidad de **PATRIC SOFT ®**, este sistema prioriza la velocidad de operación y la precisión de datos.

---

## 🚀 Características Principales

* **Gestión CRUD Completa:** Control total sobre productos (Alta, Baja, Modificación y Consulta).
* **Terminal de Venta Rápida:** Consola de comandos integrada para despachar productos mediante lenguaje natural (Ej: `3 CEMENTO`).
* **Persistencia de Datos:** Los datos se mantienen seguros en el navegador mediante `LocalStorage`.
* **Monitor de Stock Crítico:** Alertas visuales automáticas para productos con stock bajo (≤ 5 unidades).
* **Buscador en Tiempo Real:** Filtrado instantáneo de productos por nombre.
* **Interfaz Dark Premium:** Diseño ultra-moderno optimizado para reducir la fatiga visual.

---

## 🛠️ Tecnologías Utilizadas

* **HTML5 & CSS3:** Estructura semántica y diseño responsivo con variables dinámicas.
* **JavaScript (Vanilla):** Lógica de negocio pura, sin dependencias externas, garantizando máxima velocidad.
* **FontAwesome:** Iconografía profesional para una experiencia de usuario intuitiva.
* **Google Fonts (Inter):** Tipografía de alta legibilidad.

---

## ⌨️ Comandos de la Terminal

Para agilizar las ventas, abra la terminal en la esquina inferior derecha y utilice el siguiente formato:
`[CANTIDAD] [NOMBRE_DEL_PRODUCTO]`

> **Ejemplo:** `10 ARENA` (Luego presione Enter)
